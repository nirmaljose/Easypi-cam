How to watch live video?

Smartphone
- Download VLC player, if not available.
- Goto Stream option
- Enter "rtsp://{ipaddress}:8554/
Computer
Configuration
- Download VLC Player, if not available
- Goto Tools->Preference
- Select "Input/Codecs" Tab -> Enable "RTP over RTSP(TCP)"
- Click Save
- Restart the application,for first time
- Open VLC Player
- Ctrl+N or Media->Open Network Stream
- Enter Network URL as "rtsp://{ipaddress}:8554/"
